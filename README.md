## Учебный скрипт

### Это виджет, который показывает несколько последних успешных банковских операций клиента.

### Запуск:
1. Положите данные в корень проекта в файл operations.json
Пример структуры данных:   
    { \
    "id": 939719570,\
    "state": "EXECUTED",\
    "date": "2018-06-30T02:08:58.425572",\
    "operationAmount": {\
      "amount": "9824.07",\
      "currency": {\
        "name": "USD",\
        "code": "USD"\
      }

2. Запустите main.py

### Технологии:
